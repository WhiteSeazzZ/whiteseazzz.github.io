mac使用github和Hexo搭建个人博客

发表于 2018-01-08 | 分类于 技术

之前学什么记些东西总是用mac自带的标签或者各种笔记应用，而且自己也不怎么注重整理，今天去找以前的记录找了半天发现，哇！写的都是什么东西连标注都没有，看半天才知道自己需要用到的到底是啥！真是wtf！基于这种情况考虑，自己必须要有一个用于记录东西的地方，并且最好是公开的可供人阅读的（因为要给人看必须强迫自己写的很明白orz…），所以作为一个程序猿👨‍💻‍，当然要来自己动手搞一个个人博客啦！
so 作为博客第一篇，就是记录怎么搭一个博客！

创建github仓库

注册完之后我们需要一个github仓库来保存网站，点击github首页右边栏的New repository按钮来创建仓库

然后来到创建仓库的页面，此处注意 Respository name 要写成 username.github.io 的形式，这将会是你个人博客的域名，并且，username 要与前面的 Owner 保持一致！！！下边这张是因为我仓库名创建过了，只是示例。

更改配置

切换到刚刚生成的项目目录内，其中_config.yml是网站的配置文件，使用编辑器打开，以下是其中一些需要设置的参数

参数 描述
title 博客名
description 博客描述
author 你的名字
language: zh-Hans 语言要设置成 中文
theme: 主题名称
type: git 设置成git

除了上边之外还要在 type 后中添加一个参数

repo: https://github.com/username/username.github.io.git //Github仓库地址
