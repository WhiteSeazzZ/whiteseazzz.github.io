活着

发表于 2018-03-03 | 分类于 其他

他感觉自己可能已经死了

夕阳让屋里的浮尘清晰可见，它们静静地悬在那里，仿佛恒古未变，他忽然站了起来，从床头快速的走到床尾，就像一场海洋里突如其来的风暴，扰乱这一切又什么都没改变。

他也不知道自己在想些什么，许多念头一闪而过，脑子里嘈杂个不停，他想起来小时候去街里赶集，旁边此起彼伏的叫卖声，斤斤计较的女人，抱着孩子哄的男人，通通从耳朵涌向大脑，变成像棉花一样的东西充斥在脑海里。

头痛，他觉察到了，头很痛，他从床尾转过身来，准备走向床头。夕阳铺上暖黄色之后，一切事物都会显得旧迹斑斑，墙上有一些细小的裂纹格外清晰，犹如深渊沟壑。他从地上看到了自己的影子，一直被拉长到屋顶上，他觉得没由来的一阵恶心。从床尾到床头的路都像被拉长了，变成了一条一直向上一直向上的路，一直通向屋顶，又通向某些遥远的地方。有什么东西在拉扯他，眼前的所有东西，屋里的所有东西，都被拉扯成色彩斑斓的直线，他看到万千条线，在眼前汇聚成一个光点。我走不到那里了，他说。

他转过身来，走到窗前推开窗户，闭上眼任凭自己向前倒下，耳边有呼呼的风声，世界从来没有如此的清净，他想起了存在过的每一个夜晚，每一个值得被记住的事物。

我还活着。
